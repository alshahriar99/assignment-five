Important FAQ

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to manipulate HTML and XML documents programmatically.

How do you select an element from the DOM?

You can select an element using methods like document.getElementById('id'), document.querySelector('.class'), or document.getElementsByClassName('class'). These methods allow you to access elements by their ID, class, or other selectors.

What is event delegation in the context of the DOM and why is it useful?

How do you modify an element's attributes and styles using the DOM?

You can modify attributes using the setAttribute('attr', 'value') method, and for styles, you can use the style property, such as element.style.color = 'red'. You can also directly change CSS classes with element.classList.add('className') or element.classList.remove('className').
